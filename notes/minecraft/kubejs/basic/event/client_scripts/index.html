客户端脚本

304字约1分钟

2024-11-15

client_scripts 文件夹包含每次客户端资源重新加载时都会加载的脚本。可以通过快捷键 F3 + T 或者使用指令 /kubejs reload client_scripts 来对该文件夹下的脚本进行重载。
